# BeenMap
Beenmap est un projet de WebApp pour suivre en local, les pays visités à travers le monde.
L'app comprend diverses fonctionnalités comme l'ajout du pays visité par le clic sur la carte ou bien à travers la recherche. Le tout, en reprenant les données et la librarie LeafLet
